    In HTML we have formats from h1 to h6 depending on the type of heading.

    h3 heading (24px)

    h4 heading (18px)

    h5 heading (14px)
    h6 heading (12px)

      // code //

  • Lists

      We have 3 types of lists :
    • Simple List without Bullets
    • Stacked List
    • Stacked List (spaced)

      // Simple List without Bullets//

    • Sentence one
    • Sentence two
    • Sentence three

    // Code of Simple List //

    • <ul class="simple-list">
    • <li> Sentence one </li>
    • <li> Sentence two</li>
    • <li> Sentence three </li>
    • </ul>

      // Stacked List//

    • Sentence one
    • Sentence two
    • Sentence three

    // Code of Stacked List //

    • <ul class="stacked-list">
    • <li> Sentence one </li>
    • <li> Sentence two</li>
    • <li> Sentence three </li>
    • </ul>

      // Stacked List (spaced) //

    • Sentence one
    • Sentence two
    • Sentence three

    // Code of Stacked List (spaced) //

    • <ul class="stacked-list list-space">
    • <li> Sentence one </li>
    • <li> Sentence two</li>
    • <li> Sentence three </li>
    • </ul>
